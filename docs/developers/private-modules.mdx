# Private Modules

Apollo has private modules, that are access restricted extensions of the API.

These modules are only available on pre-approved servers and will not work on servers that have not received approval. These private modules are still subject to our [Acceptable Use](/apollo/acceptable-use) guidelines.

### How to Request Access

We are not currently granting broad access to private modules. However, if you have a specific use-case in mind, you may request access by contacting **apollo@lunarclient.com**.

Please include:
- Your server name & domain
- A description of your use-case
- Any relevant code snippets (if applicable)
