# Public Modules

Apollo provides a wide-range of public modules that allow you to integrate direct with Lunar Client.

These modules are available to all servers using Apollo and do not require any pre-approval. You are still expected to follow our [Acceptable Use](/apollo/acceptable-use) guidelines.
